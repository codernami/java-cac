Calculadora

 Como genero un proyecto de autor

Para ser ejecutado fuera de netbeans

desde el menù de la barra de herramientas
Run(ejecutar) selecciono Limpiar y Generar Proyecto
Abajo se despliega una ventana con el mensaje de Build succesful
Buscar la ubicación de mi proyecto
Desde la ventana izquierda(árbol de proyectos etc)
Selecciono mi proyecto Calculadora(la taza de café)
Botón derecho despliego el menú Propiedades
se abre una ventana donde en Sources veo la dirección donde está guardado mi proyecto
sabiendo la dirección ahora la busco desde C:
una vez encontrado puedo ver los archivos que se generan
En "src" se guarda mi código fuente
En "dist" se guardan los archivos de distribución
Un readme.txt y el ejecutable de mi proyecto con extensión .jar
Ejecutamos el proyecto haciendo click en Pantalla.jar
